DNX64 SDK Version History

Version 1.0.15
Release Date: 11th August 2025
£» Added GetISO and SetISO methods for controlling ISO on EdgePLUS AM4x17 and AM8x17 series.
£» Added compatibility for Edge AF4x35 series.
£» Fixed conflict issue between Edge 3.0 AM73x15 series and other cameras (built-in or external).
£» Fixed multi-camera issue with Edge 3.0 AM73x15 series in Python. 
£» Fixed compatibility issues for Premier AM3113(R4 or later), AM3111(R4 or later), and Basic AM2111(R9 or later). 
£» Fixed GetExposureValue and SetExposureValue methods for EdgePLUS AM8x17 series.

Version 1.0.11
Release Date: 06th March 2024
£» Fixed GetExposureValue zero return issue on Edge 3.0 AM73x15 series.

Version 1.0.10
Release Date: 27th December 2023
£» Provided Python API and example code on GitHub repository (https://github.com/dino-lite/DNX64-Python-API).
£» Added compatibility to Dino-Lite EdgePLUS AM8x17 series.

Version 1.0.8
Release Date: 22nd August 2022
£» Added SetLensFinePos/GetLensFinePosLimits to fine tune the lens position for devices listed below.
    1. EdgePLUS AM4917 series.
    2. Edge 3.0 AM73915MZT, revision R10 or later.
    3. Edge 3.0 AM73915MZTL, revision R10A or later.
£» Added SetAimpointLevel for models equipped with Aim Point Laser(APL) to control the laser intensity.  
£» Added compatibility to 32-bit OS.
£» Renamed GetDeviceID with GetDeviceId for compatibility to 32-bit. 
 
Version 1.0.6
Release Date: 7th April 2022
£» Added compatibility to Dino-Lite Edge 3.0 AM73x15 series.
£» Added SetEFLC to control LED configuration of models providing eFLC.
£» Replaced GetLensValueRange with GetLensPosLimits.

Version 1.0.5
Release Date: 28th October 2021 
£» Added support for controlling Dino-Lites via WF-10 or WF-20 Wi-Fi streamers. 
£» Added GetWiFiVideoCaps to return kinds of resolution, width array and height array of supported of Dino-Lites connected via Wi-Fi streamer WF-10 or WF-20. 
£» Added GetWiFiImage to capture images when using a Dino-Lite connected via Wi-Fi streamer WF-10 or WF-20.
£» Added SetWiFiVideoRes to set resolution of Dino-Lites connected via Wi-Fi streamer WF-10 or WF-20. 

Version 1.0.4
Release Date: 3th September 2021 
£» Added GetLensValueRange/SetLensPos for Dino-Lite Edge AM4915/7915 series. 

Version 1.0.2
Release Date: 8th June 2020 
£» Added SetExposureValue/GetExposureValue.
£» Added support for Dino-Lite EdgePLUS AM4x17 series.

Version 1.0.0
